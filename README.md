Техническое задание:
Написать парсер, кототрый будеn обращаться к WBAPI, получать данные о складах и отправлять их в Google Sheets

Запустить проект можно с помощью одной команды "docker-compose up --build"

НО предварительно вам нужно будет добавить .env + для корректной работы с Google Sheets необходимо создать сервисный аккаунт и загрузить его ключ credentials.json в корень проекта.
Пример структуры указан в credentials.example.json.

Технологии:
Node.js — серверная часть
Express.js — REST API
Knex.js — ORM для работы с PostgreSQL
PostgreSQL — база данных
node-cron — планировщик задач
Google Sheets API — обновление таблиц
Docker Compose — контейнеризация приложения
